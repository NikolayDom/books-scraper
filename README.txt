# Парсер каталога книг BooksToScrape

Этот скрипт автоматически собирает информацию о **всех** книгах с учебного сайта [books.toscrape.com](https://books.toscrape.com/).

## 📦 Что собирает скрипт?
- Название книги
- Цену
- Ссылку на изображение обложки

## 🚀 Как запустить

1. Убедитесь, что у вас установлен Python 3.7 или выше.
2. Установите необходимые библиотеки:
   ```bash
   pip install requests beautifulsoup4
